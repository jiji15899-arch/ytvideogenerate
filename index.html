GitHub Pages 배포
1
GitHub 리포지토리 생성
github.com → New Repository → 이름 입력 (예: yt-ai-studio) → Public → Create
2
파일 업로드
다운로드한 index.html, _headers 파일을 리포지토리에 업로드
3
GitHub Pages 활성화
Settings → Pages → Source: main branch → / (root) → Save
4
접속 확인
https://[username].github.io/[repo-name]
☁️
Cloudflare Pages 배포 (더 빠른 CDN)
1
Cloudflare 계정 & 로그인
dash.cloudflare.com → Workers & Pages → Pages → Create a project
2
GitHub 연동 또는 직접 업로드
Connect to Git → GitHub 연동 or Upload → index.html, _headers 파일 업로드
3
빌드 설정
Framework: None / Build command: 없음 / Output: / (root) → Save and Deploy
4
커스텀 도메인 (선택)
Custom domains → 도메인 추가 → DNS 자동 설정 (무료 SSL 포함)
