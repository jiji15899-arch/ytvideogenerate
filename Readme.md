## 🚀 배포 방법

### GitHub Pages (무료)
```bash
1. github.com → New Repository (Public)
2. index.html, _headers, README.md 업로드
3. Settings → Pages → Source: main → Save
4. https://[username].github.io/[repo] 접속
```

### Cloudflare Pages (더 빠름)
```bash
1. dash.cloudflare.com → Workers & Pages → Pages
2. Create project → Upload assets
3. index.html + _headers 업로드
4. Deploy → 완료 (전 세계 CDN 자동 배포)
```
